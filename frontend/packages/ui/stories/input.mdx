# Input

Bare native `<input>` styled with SICO tokens. Built on
[`@base-ui-components/react`](https://base-ui.com/react). Shadcn restyle of
`base-nova` — only styles diverge from upstream, no upstream drift.

This is the leaf primitive. For affixes (icons, prefix/suffix text, action
buttons, password toggle, clear button), compose with `<InputGroup>`. For
label, description, and error wiring, compose with `<Field>`. Do not extend
this file.

### States

`aria-invalid="true"` switches the fill to `input-fill-error` and the border
to `input-stroke-error`. The same attribute is read by `<Field>` and
`<InputGroup>` to propagate invalid state — there is no separate prop.

### Sizing & spacing

| Property      | Class            | Value |
| ------------- | ---------------- | ----- |
| Height        | `h-8`            | 32px  |
| Padding (X)   | `px-3`           | 12px  |
| Padding (Y)   | `py-1`           | 4px   |
| Radius        | `rounded-lg`     | 8px   |
| Border        | `border`         | 1px   |
| Focus shadow  | `shadow-s`       | `var(--shadow-s)` |
| Font size     | `text-sm`        | 14px  |

### File-input slot

`file:` selectors style the native upload button so it matches the input
shell. No JS, no separate component.

| Class                | Effect                                |
| -------------------- | ------------------------------------- |
| `file:h-6`           | 24px button height                    |
| `file:bg-transparent`| Transparent fill                      |
| `file:text-sm`       | 14px                                  |
| `file:font-medium`   | Medium weight                         |

## Upstream Audit

Shadcn restyle of `base-nova/input` (no upstream drift):

### Token swaps

- `border-input` → `border-input-stroke-rest`.
- `bg-background` → `bg-transparent`.
- Focus styling uses `focus-visible:border-input-stroke-pressed` +
  `focus-visible:shadow-s` (keyboard-only focus indicator, no ring).
- Invalid styling uses `aria-invalid:border-input-stroke-error` +
  `aria-invalid:bg-input-fill-error` (no ring).

### API contract

- No prop API changes — accepts all native `<input>` props.
- Logic and structure are unchanged from upstream.

### Deleted from upstream

| Deleted | Reason |
| ------- | ------ |
| `dark:` variants | SICO has no dark theme yet |
| `md:text-sm` responsive override | SICO uses 14px everywhere |
