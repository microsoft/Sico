# Tooltip

A floating label that surfaces context when a user hovers, focuses, or taps a
trigger. Built on [`@base-ui-components/react`](https://base-ui.com/react). The
dark inverted treatment is SICO-specific — every other behaviour is upstream.

### Sizing & spacing

| Property       | Class / token          | Value                          |
| -------------- | ---------------------- | ------------------------------ |
| Radius         | `rounded-lg`           | 8px                            |
| Padding        | `px-3 py-2`            | 12px / 8px                     |
| Font size      | `text-sm`              | 14px                           |
| Shadow         | `--shadow-m`           | `0 1px 2px + 0 1px 4px`        |
| Side offset    | `sideOffset` (default) | 8px                            |
| Show arrow     | `showArrow` (default)  | `true` / `false`               |

## API

| Component | Props | Description |
| --- | --- | --- |
| `Tooltip` | Radix `Tooltip.Root` props | Controls open state, delay, etc. |
| `TooltipProvider` | `delayDuration`, `skipDelayDuration` | Shared delay context for grouped tooltips |
| `TooltipTrigger` | `asChild` | The element that triggers the tooltip |
| `TooltipContent` | `side`, `sideOffset`, `showArrow`, `className` | Floating content panel |
