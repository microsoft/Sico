# Design Tokens

Every value in `@sico/ui` flows from one source of truth — the `@theme {}`
block in `globals.css`. These tables are generated from it directly, so the
page can never drift from the tokens the components actually consume.

Tokens are layered. Each layer references the one below it through `var()`,
never the reverse:

**Component → Shadcn Theme → Semantic → Brand → Foundation**

A value that resolves to another token is shown as an **alias** — follow the
chain down to the Foundation hex it ultimately paints.

---

## Foundation

Raw, context-free values. Foundation tokens are the only layer that holds
literal hex, gradient, shadow, and typography values; everything above
references them.
