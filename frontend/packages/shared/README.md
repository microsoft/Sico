# @sico/shared

Shared business logic, React hooks, atoms, components, and utilities.

## Development

```bash
npm run dev              # Start Storybook (:6007)
npm run build-storybook  # Build static Storybook
npm run test             # Run tests (Vitest)
npm run test:watch       # Watch mode
npm run test:coverage    # Coverage report
npm run lint             # Full lint pipeline (tsgo → oxlint → ESLint → oxfmt)
```
