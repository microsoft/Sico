**Platform note:** Hover, DoubleClick, and Hotkey are **desktop-only** actions (Windows/macOS). They are NOT available on mobile platforms. For mobile test cases, do not use these three actions.

| Action | Description | Example |
|--------|-------------|---------|
| Click | Click on a specified UI element | Click the "Message Copilot" input field |
| Drag | Drag an element from one position to another | Drag the slider handle to adjust a setting |
| Scroll | Scroll within a specified area in a given direction | Scroll down in the conversation area to see earlier messages |
| Type | Type text content into the currently focused input field | Type "hello" in the Message Copilot input field |
| Launch | Launch a specified application | Launch "Copilot" from the Android home screen |
| Wait | Wait for a page or element to finish loading | Wait for Copilot's response to finish streaming |
| Finished | Mark the task as completed with a summary | Finished: "Successfully sent a message and received a response" |
| CallUser | Conclude the answer for an information-retrieval question | CallUser: "The conversation title is 'Greeting and Location Inquiry'" |
| LongPress | Long press on a specified UI element | Long press a message bubble to copy or share |
| PressBack | Press the Back button or navigate back | Press Back to return to the previous screen |
| PressHome | Press the Home button | Press Home to return to the Android home screen |
| PressEnter | Press the Enter key | Press Enter to send the message |
| PressRecent | Press the Recent button to view recent apps | Press Recent to open the recent apps view |
